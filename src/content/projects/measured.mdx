<h3>Overview</h3>

I created this project to streamline comparing online clothing measurements to your own. Everyday brands usually stick to standard sizing, but niche, designer or international items vary wildly. The most reliable way to know how an item is going to fit on you is a direct measurement comparison. This eliminates all the guesswork so you know exactly how something fits.

<h3>Notable Tech Used</h3>

<h4>React</h4>

- React was used for the popup/main window
- High interactivity UI to avoid full page reloads
- Component base design for high repetitive structures like ItemCard.
- React Router to manage navigation between Home, Items, Add and Edit views
- Context API handled global wardrobe state across various routes

<h4>Web Extension API</h4>

- Content scripts to scrape data using Mutation Observer API and inject UI directly into sites (Grailed and SSENSE.)
- `chrome.storage.local` persists user wardrobe data
  across sessions
- Background service worker handles cross-origin requests and
  caching
